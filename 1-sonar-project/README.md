# 🔊 Sonar Rock vs Mine Prediction

A machine learning project that classifies sonar signal readings as either **rock** or **mine** using Logistic Regression.

## 📌 Overview
- **Algorithm:** Logistic Regression
- **Dataset:** Sonar dataset (208 samples, 60 features)
- **Task:** Binary Classification (Rock = R, Mine = M)
- **Accuracy:** ~83% on test data

## 🛠️ Tech Stack
- Python, NumPy, Pandas
- Scikit-learn
- Jupyter Notebook

## 📁 Files
- `data/sonar.csv` — Dataset used by the project
- `notebooks/sonar_rock_vs_mine_prediction.ipynb` — Jupyter notebook with full analysis
- `src/train.py` — Runnable Python training script
- `requirements.txt` — Python dependencies

## 🚀 Run
```bash
pip install -r requirements.txt
python src/train.py
```
