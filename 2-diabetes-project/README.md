# 🩺 Diabetes Prediction System

A machine learning project that predicts whether a patient is diabetic or not using a **Linear Support Vector Machine (SVM)** classifier.

## 📌 Overview
- **Algorithm:** Linear Support Vector Machine (SVM)
- **Dataset:** Pima Indians Diabetes Dataset (768 samples, 8 features)
- **Task:** Binary Classification (Diabetic = 1, Non-Diabetic = 0)
- **Accuracy:** ~77% on test data

## 🛠️ Tech Stack
- Python, NumPy, Pandas
- Scikit-learn
- Jupyter Notebook

## 📁 Files
- `data/diabetes.csv` — Dataset used by the project
- `notebooks/diabetes_prediction_svm.ipynb` — Jupyter notebook with full analysis
- `src/train.py` — Runnable Python training script
- `requirements.txt` — Python dependencies

## 🚀 Run
```bash
pip install -r requirements.txt
python src/train.py
```
